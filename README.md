# Lista Telefônica em Java

Este projeto é uma aplicação desenvolvida em Java com o objetivo de simular uma agenda telefônica, permitindo o cadastro e consulta de contatos.

## Objetivo

Praticar conceitos fundamentais de programação, como:
- Lógica de programação
- Organização de código em classes
- Manipulação de dados
- Estruturação de projetos em Java

## Funcionalidades

- Cadastro de contatos
- Consulta de contatos
- Organização dos dados em estrutura de classes
- Leitura e escrita de dados

##  Tecnologias utilizadas

- Java
- IDE IntelliJ

## Estrutura do projeto

- `Main.java` → ponto de entrada da aplicação  
- `AgendaTelefonica.java` → lógica principal  
- `ListaTelefonica.java` → gerenciamento dos contatos  
- `LeitorBuffer.java` / `EscritorBuffer.java` → manipulação de dados
